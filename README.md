# Приложение "Погода"

Одностраничное приложение без фреймворков, показывающее погоду по городу или координатам.
Используется публичный API [Open-Meteo](https://open-meteo.com/) (без ключей).

## Фичи

- Поиск погоды по **названию города**
- Поиск погоды по **широте и долготе**
- Можно добавлять **несколько виджетов** погоды
- Для виджетов по координатам отображается **карта** (OpenStreetMap, `iframe`)
- Дополнительные данные:
  - текущая температура
  - ощущается как
  - скорость ветра
  - влажность
  - местное время в локации
- Разные **иконки** в зависимости от погоды:
  - солнце, облака, дождь, снег, гроза, туман и т.д.
- Актуальная дата/время в шапке
- Удаление виджета по кнопке `×`
- Адаптивная вёрстка (карточки перестраиваются в сетку)

## Технологии

- Чистый **HTML + CSS + JavaScript**, без библиотек и фреймворков
- Работа с API через **fetch + async/await**
- Валидация пользовательского ввода на стороне клиента

## API

### Геокодирование

`GET https://geocoding-api.open-meteo.com/v1/search?name={city}&count=1&language=ru&format=json`

Используется для получения широты, долготы, страны и часового пояса по названию города.

### Погода

`GET https://api.open-meteo.com/v1/forecast?latitude={lat}&longitude={lon}&current_weather=true&hourly=temperature_2m,relativehumidity_2m,apparent_temperature&timezone=auto`

Используется для получения текущей погоды, влажности и "ощущается как".
